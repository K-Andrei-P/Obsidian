# Sorting Algorithm

A sorting algorithm solves the problem of sorting a dataset.

There are different kinds of principles to solving a sorting problem:

- [[Divide and Conquer]]
- [[Trial and Error]]
- [[Brute Force]]
- [[Greedy]]

Each of these have different amounts of time it takes to solve a problem based on the length of a dataset. This is written in the [[Big O Notation]].
