# Nodes

Node is a structure that stores data and keeps track of who it is connected to. Usually the next Node.

Node Structure:
	Node 
	{
			int number;
			Node next;
	}

The next attribute is a [[Pointer]] to the next Node.
This can be imaged to be the [[Edge]] of a Graph
