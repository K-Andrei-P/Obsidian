# Big O Notation

Big O Notation says the runtime complexity of [[Algorithms]].

It is written O(x).

Algorithms have different efficiencies.
Some are faster and some are slower.

This are different time complexities in descending order:
1. O(nˆ2)
2. O(nlogn)
3. O(n)
4. O(log n)
5. O(1)

Big O Notation is always the worst case runtime complexity.

The best case (lower bound) is written as: Ω(x)
