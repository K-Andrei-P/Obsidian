# File IO

File Input / Output is a section in programming that allows you to write and read out of external files. For example a txt file.

In [[C]], there are ready made methods from the [[Library]] stdio.h for reading and writing files:
- fopen
- fclose
- fprintf
- fscanf
- fread -> reading binary data
- fwrite
- fseek -> move around in a file

There are different operations that can be done inside files:
- reading
- writing
- appending
- reading binary
- writing binary
