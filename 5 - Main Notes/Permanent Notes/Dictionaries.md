# Dictionaries

This [[Abstract Datatype]] stores data by assigning keys -> values.

You can picture a Dictionary as an [[Array]] of [[Linked List]]s called a [[Hash Table]]. 

There are multiple steps in assigning keys to a value:
1. Key gets inputted in a [[hash function]]
2. Hash output gets searched inside the Array
3. [[Linked List]] gets searched until the matching key value pair gets found.
